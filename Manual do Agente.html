2. Visão de Hierarquia e Arquitetura do Sistema

O Software foi projetado em Python Assíncrono, seguindo a arquitetura em escala (Multi-Tenant) com "Franquias".

  • Repositório Central: O Código Mestre reside em https://github.com/ValenteArtes/leo-cloud.
  • Blueprint Instalador (render.yaml): Engatilha a importação nativa na máquina virtual Web Service do Render em 1-clique.
  • Servidor Web (main.py): Possui um Servidor HTTP Fantasma (Porta 10000) escondido numa thread paralela que valida a instância 24h por dia no plano Free ($0).
  • Inteligência (agent.py): Usa "Llama 3.1 8B Instant" (Groq) e mapeia ferramentas via Voice (Edge TTS).
  • Rastreio Ético Corporativo (telemetry.py): Toda vez que um cliente do amigo chama a ação sheet.py ou inicia conversa, um relatório particular notifica o celular do João (ID Master).
